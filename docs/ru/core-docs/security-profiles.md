# Security Profiles

TARINIO 3.0.2 включает 5 встроенных security-профилей сервиса:

1. `strict`
2. `balanced`
3. `compat`
4. `api`
5. `public-edge`

Профиль — это базовый preset. Оператор может оставить профиль и добавить собственные override-параметры.

## Назначение профилей

- `strict`: максимально жёсткий профиль для внешних сервисов.
- `balanced`: профиль по умолчанию для смешанного production-трафика.
- `compat`: снижает риск ложных блокировок для legacy-приложений.
- `api`: профиль для API-сервисов с включённой базой API Positive Security.
- `public-edge`: профиль для публичного edge-контра с усиленными anti-bot/blacklist настройками.

## Модель применения

- Выбор профиля подставляет рекомендованные параметры.
- Ручные изменения оператора сохраняются при последующих сохранениях.
- Название профиля видно в таблице сервисов и в настройках сервиса.
- В RAW-редакторе доступен `WAF_SITE_SERVICE_PROFILE`.
