# Бенчмарки

Эта страница относится к текущей ветке документации.

Бенчмарки используются для:

- проверки производительности на целевом профиле трафика;
- сравнения релизов по задержке и пропускной способности;
- контроля регрессий перед обновлением.

Рекомендуется фиксировать:

- конфигурацию стенда;
- версию TARINIO;
- профиль нагрузки;
- p50/p95/p99 задержки;
- долю блокировок и ложных срабатываний.
