# Бэкапы (RU)

Базовая версия документации: `1.0.10`

## Цель

Зафиксировать минимально безопасный путь резервного копирования для single-node развёртывания.

## Что бэкапить

- БД (PostgreSQL volume)
- volumes с артефактами ревизий и runtime state (если используется)
- `.env` (вне репозитория, хранить безопасно)

## Важно

Перед обновлением бэкап обязателен (см. `docs/ru/upgrade.md`).
