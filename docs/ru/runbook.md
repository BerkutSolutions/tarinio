# Runbook (RU)

Базовая версия документации: `1.0.4`

## Базовые проверки

- Liveness: `GET /healthz`
- Версия: `GET /api/app/meta`

## Типовые действия

- После изменения конфигурации: compile и apply новой ревизии.
- При проблемах с трафиком: откат на последнюю успешную ревизию.

## Логи

Ориентируйтесь на логи контейнеров:
- `control-plane`
- `runtime`
- `worker`
