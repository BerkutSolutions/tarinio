# Политика безопасности

Продукт: **Berkut Solutions - TARINIO**

## Поддерживаемые версии

Исправления по безопасности выпускаются для последнего релиза и `latest-1`.

## Как сообщить об уязвимости

Передавайте информацию об уязвимости приватно команде Berkut Solutions.

Укажите:
- версию продукта (`/api/app/meta`)
- детали окружения (single-node / compose, ОС)
- шаги воспроизведения и ожидаемое влияние
- логи или скриншоты (без секретов)

Не публикуйте детали уязвимости до выхода исправления.
