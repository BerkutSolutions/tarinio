# Вклад в проект

Продукт: **Berkut Solutions - TARINIO**

## Принципы

- Runtime остается полностью offline (без внешних CDN/runtime downloads).
- RU/EN документация и i18n-словарь поддерживаются синхронно.
- Изменения должны быть минимальными и согласованными с текущими паттернами.

## Рекомендуемый workflow

1. Запустите тесты: `go test -count=1 ./...`
2. Обновите документацию при изменении поведения.
3. Добавьте user-visible изменения в `CHANGELOG.md`.
