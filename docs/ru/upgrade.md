# Обновление / откат (RU)

Базовая версия документации: `1.0.12`

Минимально поддерживаемый путь обновления: `latest-1 -> latest`.

## Playbook (обязательно)

1. Preflight.
2. Backup.
3. Migrate.
4. Smoke checks.
5. Rollback decision.

Используйте внутренний release-playbook команды и стандартный preflight/backup/migrate/smoke процесс.
