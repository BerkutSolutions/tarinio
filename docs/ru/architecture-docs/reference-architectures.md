# Референсные архитектуры

Ниже перечислены базовые варианты развёртывания TARINIO v1.3.5+:

1. Одноузловой контур для стенда и пилота.
2. Контур высокой доступности control-plane.
3. Сегментированная enterprise-схема с выделенными зонами.
4. Лабораторный контур для нагрузочных и отказоустойчивых проверок.

## Компоненты стека по топологиям

| Компонент | Одноузловой | HA | Enterprise |
|---|---|---|---|
| `control-plane` | да | да (x2) | да (x2+) |
| `postgresql` | да | shared | shared |
| `vault` | да | да | да |
| `opensearch` | да | да | да |
| `redis` | нет | да | да |
| `clickhouse` | нет | нет | опционально |
| `tarinio-sentinel` | опционально | опционально | да |
