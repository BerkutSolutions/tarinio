# Набор бенчмарков

Эта страница относится к текущей ветке документации.

## Область

Публичный набор бенчмарков в `deploy/compose/ha-lab` предназначен для воспроизводимых локальных измерений без перегруза рабочей станции.

Он отвечает на три вопроса:

- насколько быстро стек отвечает под нормальным трафиком;
- как он ведёт себя на защищённых конечных точках, когда включаются лимиты;
- остаются ли кластер `control-plane` в режиме высокой доступности и контур наблюдаемости стабильными во время прогона.

## Сценарии

Набор бенчмарков запускает:

- базовый трафик по обычной странице арендатора;
- сценарий с ограничением частоты запросов для `tenant-01`;
- проверки доступности API через `api-lb`;
- опциональный запрос к Prometheus для проверки доступности контура наблюдаемости.

## Что сохраняется

`benchmark-http.sh` возвращает JSON с:

- общим количеством запросов;
- распределением по кодам ответа;
- средней задержкой;
- минимальной и максимальной задержкой;
- `p50`, `p95`, `p99`.

`benchmark-pack.sh` собирает всё в один итоговый JSON-документ.
