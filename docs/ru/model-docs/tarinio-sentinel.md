# TARINIO Sentinel

Эта страница относится к текущей ветке документации.

Sentinel принимает поток событий и формирует адаптивные решения для защиты периметра.

Модель учитывает:

- интенсивность источника;
- характер ошибок (`403`, `429`, `444`);
- повторяемость подозрительных путей;
- распределённость нагрузки.

Результат используется в runtime для мягкой эскалации и блокировки явных атакующих источников.
