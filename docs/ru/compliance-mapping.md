# Карта соответствия требованиям

Эта страница относится к текущей ветке документации.

## Назначение

Эта страница помогает security и compliance командам понять, как возможности TARINIO соотносятся с типовыми классами требований.

## Важное уточнение

Это functional mapping, а не заявление о формальной сертификации.

## OWASP-ориентированное соответствие

TARINIO помогает в областях:

- request inspection и policy enforcement;
- traffic visibility и event correlation;
- безопасный rollout security changes;
- rate limiting и hostile traffic suppression.

## Operational security alignment

TARINIO помогает поддерживать контрольные меры вокруг:

- least-privilege administration;
- auditability of changes;
- backup и recovery planning;
- protected administrative access;
- observable security operations.
