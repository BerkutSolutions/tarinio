# Docker Baseline

Этот раздел предназначен для базовых Docker-сценариев lab-контура.

План наполнения:

1. единый стартовый сценарий подъема (`up`);
2. минимальный smoke-check после запуска;
3. teardown сценарий (`down`) с сохранением диагностических артефактов.

Текущий статус: только базовая структура (подготовка).
