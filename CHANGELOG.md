## [1.2.5] - 24.06.2026

### Ядро
- Исправлен healthcheck `/healthcheck`: probe запросов больше не валит API дашборда ответом `502`, если runtime request telemetry временно недоступен или деградировал.
- Для management-site hostname маршруты `/api/*` теперь проксируются в `control-plane:8080`, а не в UI upstream. Это устраняет `504` и `upstream timed out` для `/api/dashboard/stats`.

### Безопасность
- Убран лишний `set $waf_site_id` из site nginx template, чтобы healthcheck и прямые запросы по IP или невалидному `Host` не создавали warning `using uninitialized "waf_site_id" variable while logging request`.
- `golang.org/x/crypto` обновлён до `v0.52.0`, чтобы закрыть Trivy code-scanning alerts и разблокировать `security-supply-chain`.

### Сертификаты
- Ручное ACME-продление теперь передаёт в backend параметры из формы, включая email аккаунта.
- Синхронизация easy-site profile теперь сохраняет `acme_account_email`.
- Автопродление сертификатов включено по умолчанию и использует ACME-настройки из профиля сайта.

### UI
- Тихое автообновление дашборда теперь помечает polling статистики как background request, поэтому временные `401`/`502`/`503`/`504` переводят запросы в backoff вместо постоянного повторного опроса из фоновой вкладки.
- Детализация по атакам и заблокированным атакам на дашборде теперь использует тот же 24-часовой источник данных, что и карточки метрик.
- При автообновлении виджетов дашборда сохраняется позиция скролла в прокручиваемых списках и блоках.
