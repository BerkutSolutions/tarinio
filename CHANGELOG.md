## [1.2.0] - 14.05.2026

### Ядро
- Исправлена ошибка в runtime NGINX: `using uninitialized "waf_site_id" variable while logging request`.
- Добавлен безопасный fallback для `waf_site_id` в шаблоне runtime-конфига, чтобы предупреждение не возникало на healthcheck и fallback-запросах.

### Рефакторинг
- Завершен этап декомпозиции ключевых монолитов UI, backend и runtime на профильные модули с тонкими фасадами.
- Закрыт крупный блок рефакторинга страниц UI (`sites`, `dashboard`, `bans`, `settings`, `administration`) и основных сервисных/рантайм-монолитов.

### Релизные скрипты
- Исправлен пункт `5` в `scripts/release.ps1`: после синхронизации версии скрипт возвращается в меню выбора режима.
- Исправлен `scripts/update-version.ps1`: меняется только версия приложения (`AppVersion`), без изменения зависимостей и lock-файлов пакетов.

### Зависимости и безопасность
- Устранены high-уязвимости в npm-зависимостях (`npm audit fix`), итоговая проверка: `0 vulnerabilities`.
